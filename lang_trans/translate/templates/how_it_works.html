How it works

Mazda Language Translator is web based software that helps you translate from one language to another. The following steps will guide you on how to use the software.

{% comment %}
{% endcomment %}

  1. Navigate to the Home page of the software using this link .

  2. Select the language to translate and the language to translate to.

  3. Input the sentence or word to be translated and see the translation happen in realtime. Remember to put space every after a word for succesful translation.

  4. You can interchange the languages to be translated using the double arrows in between the languages.
